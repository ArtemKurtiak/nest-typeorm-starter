## How to get started?

```bash
$ git clone git@github.com:ArtemKurtiak/nest-typeorm-starter.git

$ yarn
```

<p>Copy <code>.env-example</code> to <code>.env</code> file and fill template with your parameters</p>

```bash
# development
$ yarn start

# watch mode
$ yarn start:dev

# build application for production
$ yarn build

# production mode
$ yarn start:prod
```
